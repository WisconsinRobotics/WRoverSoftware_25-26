# Installation

Steps:
    1. Determine your processor architecture x86 or ARM 64
    2. Install Ubuntu 24.04.3 LTS version [here](https://cdimage.ubuntu.com/daily-live/20240421/) and choose
        appropriate image based on your architecture 
    3. Install Oracle Virtual Box on official [website](https://www.virtualbox.org/)
    4. Navigate to Oracle Virtual Box
    5. Press New button on top
    6. In a virtual machine name and operating system section, indicate vm name and location of the ISO Image (Ubuntu 24.04)
    7. Setup username and password in set up unattended guest OS installation section
    8. For specifying virtual hardware, select half of your available RAM to virtual machine and half of your cpu count.
    9. For specify virtual hard disk, specify 64 GB or more.
